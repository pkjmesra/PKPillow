:py:mod:`~PKPIL.ImageTk` Module
=============================

The :py:mod:`~PKPIL.ImageTk` module contains support to create and modify Tkinter
BitmapImage and PhotoImage objects from PKPIL images.

For examples, see the demo programs in the Scripts directory.
